# docker-selenium-python

Selenium on docker-compose.

## Build

~~~
docker-compose up --build
~~~

## Install package

~~~
docker-compose run --rm app pipenv install PACKAGE_NAME
~~~

## Run App with Selenium client

~~~
docker-compose up app
~~~

## Start

~~~
docker-compose up
~~~

## Stop & delete

~~~
docker-compose down
~~~
